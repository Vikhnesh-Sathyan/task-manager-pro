
A full-stack Task Manager application.

## Tech Stack
- Frontend: ReactJS
- Backend: Node.js, Express
- Database: SQLite
- Authentication: JWT

---

## Features
- User Registration & Login
- JWT Authentication
- Create, Read, Update, Delete Tasks
- User-specific tasks
- Task status (Pending / Completed)
- Protected APIs
- Confirmation before delete
- Logout functionality
